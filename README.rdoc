== README

Working example of endless scroll implemented with the <tt>will_paginate</tt> gem for a simple CRUD app.

<tt>git clone https://github.com/KelseyDH/will_paginate_endless_scroll_example.git</tt>
<tt>cd will_paginate_endless_scroll_example</tt>
<tt>bundle install</tt>
<tt>rake db:migrate</tt>
<tt>rake db:seed</tt>
<tt>rails s</tt>

Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
